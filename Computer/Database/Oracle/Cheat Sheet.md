Find indexes and their columns
```sql
SELECT A.INDEX_OWNER, A.TABLE_NAME, A.INDEX_NAME, B.UNIQUENESS, A.COLUMN_NAME, A.COLUMN_POSITION, A.DESCEND
FROM ALL_IND_COLUMNS A INNER JOIN ALL_INDEXES B ON A.INDEX_OWNER = B.OWNER AND A.INDEX_NAME = B.INDEX_NAME
WHERE A.INDEX_OWNER LIKE UPPER('%') AND A.TABLE_NAME LIKE UPPER('%')
ORDER BY A.INDEX_OWNER, A.TABLE_NAME, A.INDEX_NAME, A.COLUMN_POSITION;
```
